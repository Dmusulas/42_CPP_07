# 42 C++ Module 07

This module introduces templates, enabling generic programming.

## Concepts
- Function templates.
- Class templates.
- Template specialization.

## Exercises
- **ex00 (Start with a few functions):** Implementing generic `swap`, `min`, and `max` functions.
- **ex01 (Iter):** A generic `iter` function that applies a function to every element of an array.
- **ex02 (Array):** Creating a generic `Array<T>` class that handles memory and out-of-bounds exceptions.

## Usage
```bash
cd ex02
make
./templates
